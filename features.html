Gorp's Core Features

These features are the bread and butter of Gorp. These are the features I spend the most time on improving, fixing, and expanding upon.

Instance & World Management

Start & Stop Start, stop, and interact with running server instances.
Manage Instances Create and delete server instances as required.
Manage Worlds Create, delete, and switch worlds in your server instances.

Backups & Restoring

Backup Worlds Backup the currently active world files in an instance.
Restore a Backup Restore a particular instance world backup.

Updating & Upgrading

JAR Update Utility Update the default global JAR file to the latest stable version.
Gorp Upgrade Utility Upgrade your Gorp installation to get features as they come out.
