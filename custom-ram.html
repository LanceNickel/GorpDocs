Custom RAM Allocation

It is important to set the right amount of RAM for an instance if you will use it differently than what the default value provides (4G). If you are running multiple instances, customizing RAM per instance is recommended to properly distribute your computer's resources.

Learn more about server resources

Setting Custom RAM

  1. Go to the instance directory, server instances are located in /minecraft/servers/.
  2. Edit the run.sh file using your favorite text editor (such as Vim).
  3. Change the RAM option to be the desired amount, followed by a 'G' immediately after.
    for example: 8G
  4. Save your changes and quit the text editor.
  5. Start the instance with gorp start [name]
