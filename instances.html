Creating & Deleting Server Instances

Create an Instance

sudo gorp create-server [name]

This creates a new server instance with the name given. The instance's directory will be created at /minecraft/servers/[name].

While creating, the instance will confirm that you agree to the Minecraft End User License Agreement. It will also confirm your agreement that Gorp is for personal use only.

After running this, the instance is created! Run sudo gorp start [name] to start the instance. The first startup will take a bit longer than usual as some first-time setup needs to be done.

Delete an Instance

sudo gorp delete-server [name]

PLEASE make sure to back up this instance before you delete it: sudo gorp backup [name]

Deleting server instances is irreversible. Please ensure to backup any worlds that are in the instance directory.
