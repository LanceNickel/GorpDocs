Backup Worlds

sudo gorp backup [name]

Gorp has a backup utility that will compress and copy an instance's world files to a backup destination. By default, the destination is on the local machine, but it is highly recommended to set the destination to another machine. The backup destination can be customized in the Gorp configuration file.

The backup utility has been designed to work ONLY with Paper servers.

Instances With Multiple Worlds

On instances with multiple worlds, the currently active world will be the one backed up. You can switch the active world for a given instance using the world management tools.
