Creating, Deleting, & Switching Worlds

Create a World

sudo gorp create-world [server] [name]

This creates a new world in the instance you specified. The new world will not actually be created until you run the server. This world save is independent of the other worlds created for this instance.

World names may NOT have spaces or special characters in them.

Delete a World

sudo gorp delete-world [server] [name]

This deletes the world in the instance you specified. This action is not recoverable, please take a backup!

Switch Worlds

sudo gorp switch-world [server]

This command does not need a world name. When run, it will provide a list of options for you to choose from.
