Gorp's Self-Upgrade Utility

gorp upgrade

Gorp has been created to allow for seamless and easy self-upgrading. This way, when new features and options are added, you can get them to your installation with a single command.

Important Notes

  • Gorp clears all files before installing the new ones. This includes any file in Gorp directories, not just Gorp scripts.
  • Options set in gorp.conf and each server's run.sh are preserved.
