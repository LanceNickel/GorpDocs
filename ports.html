Unique Server Ports

When running multiple server instances at the same time, it is important that each instance is running on a unique port number. Not doing so will cause errors and may even make your servers unavailable.

Chaning an Instances's Port Number

Caution: Please change port number configurations with the server instance STOPPED!

The port settings for an individual instance is located in its server.properties file. The server-port option will be 25565 by default.

Change this option using your favorite text editor (like Vim) to any value you like. Generally, counting up from 25565 is fine (25566, 25567, etc.)

No other configuration is needed, Gorp will pick up on the port change.
