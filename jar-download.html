Server JAR Download Utility

sudo gorp get-jar [version]

Gorp includes a JAR download utility, allowing you to download the Paper JAR file for any version of Minecraft since 1.8.8. This makes downloading a custom version of Minecraft for a particular server easier than manually downloading the JAR file from the Paper website.

Intended Use

This feature is simply a utility: It just downloads JAR files. You need to customize individual server instances to use your downloaded JAR file. This utility will not update these JAR files, and downloaded JAR files will not receive updates when sudo gorp update is run.

If you want to change all servers to a certain Minecraft version, edit the Gorp configuration file. This ensures that not only do all servers use your desired version, but they receive software updates when the Gorp updater is run.

Attention: Paper only supports the latest version of Minecraft. Any older Paper JAR files used will not receive regular updates from the Paper developers. In some cases, this is a security risk.
