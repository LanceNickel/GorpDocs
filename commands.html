Gorp Command Reference

The command structure of Gorp is: gorp [command] [server name] [arguments]

Start & Stop Instances see: start & stop

Command Description
gorp start [server name] Starts an instance.
gorp stop [server name] [arguments] Stops an instance. By default, takes a backup and gives a warning.
Arguments accepted:
  • fast: Skips the backup
  • now: Skips both backup and warning. Immediate shutdown.
  • power: Same as a fast shutdown, however the warning specifies the reason for the shutdown is due to a power outage.
gorp restart [server name] [arguments] Restarts an instance. By default, takes a backup and gives a warning.
Arguments accepted:
  • fast: Skips the backup
  • now: Skips both backup and warning. Immediate restart.

Create & Delete Instances see: manage instances

Command Description
gorp create-server [server name] Creates a new server instance with the passed name.
The instance is created in /minecraft/servers/[name].
gorp delete-server [server name] Deletes the given server instance's files. This action is not reversible. Be sure to take a backup and double-check which instance you want to delete.

Create, Delete, & Switch Worlds see: manage worlds

Command Description
gorp create-world [server name] [world name] Creates a world with the given name in the specified server. The world will be created next time the server instance is started.
gorp delete-world [server name] [world name] Delets a world with the given name in the specified server. This action is not reversible. Be sure to take a backup and double-check which world you want to delete.
gorp switch-world [server name] Presents you a list of worlds to choose from. Choose the world by entering its name as it appears in the list of options.

Backup & Restore Worlds see: backups

Command Description
gorp backup [server name] Backs up the world files for the given server instance. The currently active world will be the one to be backed up. Learn more about this behavior.
gorp restore [server name] Presents you with options of what world you wish to restore, along with specifying the backup's date and exact file.

Update Server JAR see: jar updater

Command Description
gorp update Download the latest stable Paper JAR file, install it, and set it as the default JAR file.

Update Gorp see: gorp self-upgrade

Command Description
gorp upgrade Upgrades the installation of Gorp to the latest available version.
Caution: Updating Gorp will overwrite all shell script files and the run.sh file, except for where changes are anticipated.
