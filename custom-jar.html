Custom Server JAR

Important Security Notice: By setting a custom JAR at the server instance level, the instance will no longer receive the latest JAR file when gorp update is run. This can put you at risk of potentially running out of date and vulnerable software. Furthermore, make sure you trust the developer who created the JAR you are about to run.

Important Support Notice: Gorp is written for Paper MC alone. Using a custom server JAR may cause certain features, like world backups, to not work as expected.

Installing Custom JAR

Download your JAR file

  1. Download your JAR file using a command like wget. Or, use the Gorp JAR download utility.
  2. Move the JAR file to /minecraft/jars/

Configure the instance

  1. Go to the instance directory, server instances are located in /minecraft/servers/.
  2. Edit the run.sh file using your favorite text editor (such as Vim).
  3. Change the JAR= option to be the absolute path to where you stored your JAR file.
    for example: /minecraft/jars/custom.jar
  4. Save your changes quit the text editor.
  5. Start the instance with gorp start [name].
