Updating and Upgrading

Update the default JAR file used by servers in one move. Then, do the same for upgrading your installation of Gorp.

JAR Update Utility Update the default global JAR file to the latest stable version.
JAR Download Utility Download the Paper JAR files for Minecraft >1.8.8.
Gorp Upgrade Utility Upgrade your Gorp installation to get features as they come out.
