Server JAR Update Utility

sudo gorp update

Gorp makes keeping your server instances on the latest version of Minecraft a breeze! The updater gets the latest stable Paper build and installs it for all instances. By default Gorp will use the latest version of the game, at this time 1.19.3. This can be configured in the Gorp configuration file.

Also, for some special cases where you might not want a particular server instance to use the global default, the instance can be configured to use a custom JAR file. This allows all instances to be up to date, while keeping one on a custom version of Minecraft (like a retro server).
