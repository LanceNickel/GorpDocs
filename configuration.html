Gorp's Configuration File

The configuration file is located at /minecraft/gorp.conf and contains all the global options needed to make Gorp run. As new features are added, more options may be added to this file.

Only options that make sense to apply on a global scope will be added to this file. Things like the Minecraft version to use when updating and default RAM allocations live here.

Below are the options currently in the configuration file.

Option Description Expected Value Default Value
GAMEVER Minecraft game version. Used by the JAR update utility when looking for the latest JAR build. This only affects server instances where the JAR file was not overwritten. Learn about JAR overriding. See list 1.19.3
RAM Amount of RAM to be allocated to each server instance. This can be overridden on a per-instance basis. Integer followed by 'G' 4G
DEST Destination for world backups generated by Gorp. More about backups. Absolute path to destination /minecraft/backups
